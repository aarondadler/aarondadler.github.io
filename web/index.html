Projects:

I am currently working on three projects at BBN:
  • TASBE (A Tool-chain to Accelerate Synthetic Biology) IRAD: A short-term BBN-funded continuation of the TASBE project described below but with a focus on creating web service versions of the tools developed in TASBE. See synbiotools.bbn.com. My work includes using Octave, Drupal, PHP, and Python to create the online versions of the tools.
  • MADV: Morphogenetically Assisted Design Variation. MADV is a joint project with iRobot funded by DARPA DSO. MADV seeks to simplify robot redesign using morphogenesis. I am a co-PI on this project.
  • BBAC (Behavior-Based Access Control): BBAC is a security project with Drexel University funded by AFRL. I am currently designing and implementing a URL classified to help identify possible malicious behavior.
  • TORQUE: TORQUE is a Quantum Computer project sponsored by IARPA. My work in this project involves helping to refine QuaFL, a quantum computing programming language, and converting quantum algorithm specifications into QuaFL.
My previous projects at BBN include:
  • TASBE: A Tool-chain to Accelerate Synthetic Biology. TASBE was a joint project with MIT and BU, funded by DARPA. My work on the project included: compiler optizations, robotic automation, and DNA characterization.
  • Advanced Protective Services (APS). APS is an AFRL funded project to develop a protection and containment system for web services which we call a Crumple Zone. I helped design and develop components for the Crumple Zone.
  • Plan Order Induction by Reasoning from One Trial (POIROT). I worked on the XPLAIN and IODA components of the POIROT integrated learning project. The goal of the POIROT project is to learn how to execute a sequence of web service calls from a single demonstration trace by combining various machine learning algorithms. The IODA component analyses the data dependencies in the trace and the XPLAIN component uses generic rules to form groups of related steps and identify user actions such as saving or submitting data.
My Ph.D. thesis centered on constructing multimodal interactive dialogues: combining speech recognition and sketch recognition for user input and generating speech and sketching for multimodal computer output. MIDOS (Multimodal Interactive DialOgue System) helps the user describe simple mechanical (Rube-Goldberg-like) devices by asking questions. A qualitative physics simulator is used to generate possible questions to ask the user. At MIT CSAIL I was part of the design rationale group (now the Multimodal Understanding Group) led by Professor Randy Davis. Prior to that I did research with the aire group, whose focus is pervasive computing applications for intelligent spaces.
